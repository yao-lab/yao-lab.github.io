There are two main files: "main.ipynb" and "eight_predictors.ipynb".
To run the code, please follow the following instructions.

1. Download the dataset from the link "https://dachxiu.chicagobooth.edu/download/datashare.zip".
2. run the codes in file "eight_predictors.ipynb", to generate a complete datasets with eight predictors.
3. run the codes in file "main.ipynb" step by step to get the results.
