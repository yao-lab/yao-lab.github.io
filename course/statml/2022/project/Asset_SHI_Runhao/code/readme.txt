1. dataclean.R and dataclean.ipynb are used to clean raw data. The data we used are from Dacheng Xiu's website and Amit Goyal's website.
2. enet_code.ipynb, gbrt_code.ipynb, nn_code.ipynb, ols_code.ipynb, ols-3_code.ipynb, pcr_code.ipynb, pls_code.ipynb, rf_code.ipynb are the implemented machine learning methods for estimating excess returns.
3. analysis.ipynb is the code to analyze the results of the above machine learning methods.
